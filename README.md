# Forces Academy LMS (Learning Management System)

This repository contains the ongoing development of the Forces Academy LMS, built during the Code Saviours (SMC-PRIVATE) Limited ML/AI Internship Programme (Batch SI-26) under the Full Stack Track.

Weekly Progress:

Week 1: Project Setup + Database Design + Login System
1- Environment Setup: Configured local development server using XAMPP (Apache & MySQL).
2- Database Architecture: Designed and implemented relational tables (`students`, `admins`, `courses`, `notices`) in phpMyAdmin.
3- Backend Logic: Built a secure student registration and session-based login system using PHP prepared statements and password hashing (`password_hash`).
4- UI/UX Design: Implemented a clean, fully responsive Bootstrap 5 interface featuring semantic SEO elements and a structured workspace layout.
